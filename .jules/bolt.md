## 2024-05-24 - Zustand store selections | **Learning:** Codebase-specific performance anti-pattern: Using `useGameStore()` without selectors subscribes the component to the entire global state, causing massive re-renders on frequent updates like passive particle generation. Always use specific state selectors. | **Action:** Optimize Zustand store selections by using direct selectors or `useShallow` from `zustand/react/shallow`.
