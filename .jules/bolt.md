## 2026-04-15 - Zustand Store Refactor | **Learning:** Codebase-specific performance anti-pattern: Using `useGameStore()` without selectors subscribes the component to the entire global state, causing massive re-renders on frequent updates like passive particle generation. Always use specific state selectors. | **Action:** Use direct state selectors for single variables, and `useShallow` from `zustand/react/shallow` when extracting multiple variables to optimize rendering.
