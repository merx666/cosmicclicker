## 2024-06-25 - Zustand useGameStore Anti-Pattern | **Learning:** Using `useGameStore()` without a selector subscribes the component to the ENTIRE global state, causing massive re-renders on every single state update (like passive particle generation). | **Action:** Always use specific state selectors (`useGameStore(state => state.property)`) or `useShallow` for multiple properties to prevent performance degradation from unnecessary re-renders.
