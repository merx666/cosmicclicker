## Summary

Glowny opis projektu Cosmic Clicker - gry idle zintegrowanej z ekosystemem Worldcoin i Telegramem.

## Key Claims

- Projekt uzywa Next.js 16 (App Router), React 19, Tailwind CSS 4, Framer Motion i Zustand.
- Baza danych to self-hosted PostgreSQL na serwerze produkcyjnym.
- Autoryzacja i platnosci odbywaja sie przez World ID MiniKit.
