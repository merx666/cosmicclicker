# World ID

System tozsamosci i uwierzytelniania dostarczany przez Worldcoin. W wersji 4.0 pozwala na weryfikacje za pomoca presetu orbLegacy oraz mechanizmu Session Proofs. W projekcie Cosmic Clicker uzywany jest do autentykacji uzytkowników bez koniecznosci podawania hasla, opierajac sie na unikalnym nullifierze.
